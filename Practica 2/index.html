102 etiquetas HTML

En esta página se explican 102 etiquetas HTML junto con su descripción y un ejemplo de uso.

Tipo de documento

  • <!DOCTYPE>

    Indica al navegador web la versión de HTML utilizada en el documento.

    Ejemplo:

    <!DOCTYPE html>

  • <html>

    Es la etiqueta raíz de un documento HTML y se utiliza para envolver todo el contenido HTML de una página

    Ejemplo:

    <html> code </html>

Etiquetas de texto

  • <p>

    Define un párrafo de texto.

    Ejemplo:

    <p>Este es un párrafo de texto.</p>

    Este es un párrafo de texto.

  • <h1> - <h6>

    Define encabezados de diferentes niveles (del 1 al 6).

    Ejemplo:

    <h1>Este es un encabezado principal</h1> <h2>Este es un encabezado secundario</h2> <h3>Este es un encabezado de tercero</h3> <h4>Este es un encabezado de cuarto</h4> <h5>Este es un encabezado de quinto</h5> <h6>Este es un encabezado de sexto</h6>

    Este es un encabezado principal

    Este es un encabezado secundario

    Este es un encabezado de tercero

    Este es un encabezado de cuarto

    Este es un encabezado de quinto
    Este es un encabezado de sexto

  • <b> y <strong>

    Ambos definen texto en negrita.

    Ejemplo:

    <b>Este texto está en negrita</b>
    Este texto está en negrita

  • <i> y <em>

    Ambos definen texto en cursiva.

    Ejemplo:

    <i>Este texto está en cursiva</i>
    Este texto está en cursiva

  • <u> y <s>

    Definen texto subrayado y tachado, respectivamente.

    Ejemplo:

    <u>Este texto está subrayado</u>
    Este texto está subrayado

  • <area>

    define un área dentro de un mapa de imagen (un mapa de imagen es una imagen con áreas en las que se puede hacer clic).

    Ejemplo:

    <area>informacion de area>

  • <br>

    Inserta un salto de línea.

    Ejemplo:

    <p>Este es un texto.<br> Este es otro texto.</p>

    Este es un texto.
    Este es otro texto.

  • <hr>

    Inserta una línea horizontal.

    Ejemplo:

    <hr>

Etiquetas de enlaces e imágenes

  • <a>

    Define un enlace a otra página web.

    Ejemplo:

    <a href="https://www.ejemplo.com">Este es un enlace</a>
    Este es un enlace

  • <img>

    Inserta una imagen en la página web.

    Ejemplo:

    <img src="ruta/a/la/imagen.jpg" alt="Texto alternativo">

Etiquetas de listas

  • <ul>

    Define una lista no ordenada.

    Ejemplo:

    <ul> <li>Elemento 1</li> <li>Elemento 2</li> </ul>

  • <ol>

    Define una lista ordenada.

    Ejemplo:

    <ol> <li>Elemento 1</li> <li>Elemento 2</li> </ol>

  • <li>

    Define un elemento de lista dentro de <ul> o <ol>.

    Ejemplo:

    <ul> <li>Elemento 1</li> <li>Elemento 2</li> </ul>

  • <dl>

    Define una lista de definiciones (término y descripción).

    Ejemplo:

    <dl> <dt>Término</dt> <dd>Descripción</dd> </dl>
    Término
    Descripción

  • <dt>

    Define el término dentro de <dl>.

    Ejemplo:

    <dl> <dt>Término</dt> <dd>Descripción</dd> </dl>
    Término
    Descripción

  • <dd>

    Define la descripción dentro de <dl>.

    Ejemplo:

    <dl> <dt>Término</dt> <dd>Descripción</dd> </dl>
    Término
    Descripción

Etiquetas de formulario

  • <form>

    Define un formulario para recibir datos del usuario.

    Ejemplo:

    <form action="procesar.php" method="post"></form>

  • <input>

    Define un campo de entrada para recibir información del usuario.

    Ejemplo:

    <input type="text" name="nombre">

    Ingrese un texto

  • <textarea>

    Define un campo de texto multilínea para recibir información del usuario.

    Ejemplo:

    <textarea name="comentarios"></textarea>

    Ingrese un texto largo

  • <select>

    Define una lista desplegable para que el usuario seleccione una opción.

    Ejemplo:

    <select name="opciones"> <option value="1">Opción 1</option> <option value="2">Opción 2</option> </select>

  • <option>

    Define una opción dentro de una lista desplegable (<select>).

    Ejemplo:

    <select name="opciones"> <option value="1">Opción 1</option> <option value="2">Opción 2</option> </select>

  • <button>

    Define un botón.

    Ejemplo:

    <button type="submit">Enviar</button>

    Este es un boton

  • <label>

    Define una etiqueta para un campo de entrada (<input>) o para una lista desplegable (<select>).

    Ejemplo:

    <label for="nombre">Nombre:</label><input type="text" name="nombre" id="nombre">

    Campo de entrada

Operaciones Matematicas

  • <math>

    Se utiliza para escribir una sola fórmula matemática

  • <mrow>

    Se utiliza para agrupar subexpresiones, que suelen contener uno o más operadores con sus respectivos operandos

  • <msup>

    Se utiliza para adjuntar un superíndice a una expresión.

  • <mi>

    Indica que el contenido debe representarse como un identificador, como nombres de funciones, variables o constantes simbólicas.

  • <mo>

    Representa un operador en un sentido amplio. Además de los operadores en sentido matemático estricto, este elemento también incluye "operadores" como paréntesis, separadores como coma y punto y coma, o barras de "valor absoluto".

  • <mn>

    Represents a numeric literal which is normally a sequence of digits with a possible separator (a dot or a comma).

    • Ejemplo general:

    <math> <mrow> <mrow> <msup> <mi>x</mi> <mn>2</mn> </msup> <mo>+</mo> <msup> <mi>y</mi> <mn>2</mn> </msup> </mrow> <mo>=</mo> <msup> <mi>z</mi> <mn>2</mn> </msup> </mrow> </math>

    x 2 + y 2 = z 2

  • <merror>

    El elemento se utiliza para mostrar contenidos como mensajes de error

    Ejemplo:

    <math display="block"> <mfrac> <merror> <mtext>Syntax error: \frac{1}</mtext> </merror> <mn>3</mn> </mfrac> </math>
    Syntax error: \frac{1} 3

  • <mfrac>

    Se utiliza para mostrar fracciones. También se puede usar para marcar objetos similares a fracciones, como coeficientes binomiales y símbolos de Legendre.

    Ejemplo:

    <math display="block"> <mfrac> <mrow> <mn>2</mn> </mrow> <mrow> <mn>3</mn> </mrow> </mfrac> </math>
    2 3

  • <mover>

    Se utiliza para añadir un acento o un límite sobre una expresión.

    Ejemplo:

    <math display="block"> <mover accent="true"> <mrow> <mi>x</mi> <mo>+</mo> <mi>y</mi> <mo>+</mo> <mi>z</mi> </mrow> <mo>⏞<!--Llave arriba de la formula--></mo> </mover> </math>
    x + y + z ⏞

  • <mphantom>

    Se representa de forma invisible, pero las dimensiones (como la altura, el ancho y la posición de la línea de base) aún se mantienen.

    Ejemplo:

    <math display="block"> <mrow> <mi>x</mi> <mo>+</mo> <mphantom> <mi>y</mi> <mo>+</mo> </mphantom> <mi>z</mi> </mrow> </math>
    x + y + z

  • <mroot>

    Se utiliza para mostrar raíces con un índice explícito

    Ejemplo:

    <math display="block"> <mroot> <mi>x</mi> <mn>3</mn> </mroot> </math>
    x 3

  • <msqrt>

    Se utiliza para mostrar raíces cuadradas (no se muestra ningún índice)

    Ejemplo:

    <math display="block"> <msqrt> <mi>x</mi> </msqrt> </math>
    x

  • <mstyle>

    Se utiliza para cambiar el estilo de sus hijos.

    Ejemplo:

    <mstyle displaystyle="false" mathcolor="teal"> //codigo para operacion matematica </mstyle>

    Ejemplo de mstyle

    ∑ I ∏ i = 1 N

  • <mtext>

    Se utiliza para representar texto arbitrario sin significado notacional, como comentarios o anotaciones.

    Ejemplo:

    <mtext>/* comment here */</mtext>
    Theorem of Pythagoras

    dato3 dato4
